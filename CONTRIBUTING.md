# Contributing

Thank you for helping improve PARALLAX/RLAC.

Good contributions include:

- clearer prompt wording
- safer refusal-and-redirection patterns
- better behavior tests
- translations
- deployment notes for specific AI tools
- examples that do not contain harmful operational details
- evaluation rubrics

Please avoid:

- exploit prompts
- harmful transcripts
- instructions for illegal activity
- CBRN, weapons, cyber abuse, fraud, or self-harm enablement
- private or identifying information

## Contribution checklist

Before opening a pull request:

- [ ] The contribution helps safety, clarity, stability, or usability.
- [ ] It does not include reproduction keys or harmful operational details.
- [ ] It preserves user agency.
- [ ] It does not turn normal safe topics into over-refusal.
- [ ] It includes safe alternatives when discussing refusal behavior.
