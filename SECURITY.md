# Security Policy

## Supported Materials

This repository contains public safety-governance prompts, behavioral checklists, examples, and documentation for PARALLAX/RLAC.

## Reporting a Safety Issue

If you find that a prompt, skill, example, or test case in this repository can be used to enable harmful misuse, please report it responsibly.

Please include:

- the affected file
- a short description of the issue
- the type of risk involved
- a minimal, redacted proof if necessary
- suggested mitigation, if available

Do not include full exploit prompts, harmful operational instructions, CBRN details, malware, credential theft procedures, or reproducible abuse transcripts in public issues.

## Public Issue Guidance

Safe to open publicly:

- typo or clarity fixes
- non-sensitive improvement suggestions
- safe test additions
- documentation improvements

Do not open publicly:

- working jailbreak keys
- harmful red-team transcripts
- illegal synthesis details
- exploit chains
- real target information
- private data
- self-harm exploitation patterns

Use private contact or staged disclosure for sensitive issues.
