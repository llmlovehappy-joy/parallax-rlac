# Responsible Disclosure

PARALLAX/RLAC is intended to reduce harmful AI behavior, not to publish exploit techniques.

If you discover a vulnerability in a commercial model, local model, agent framework, or deployment using these materials:

1. Preserve evidence privately.
2. Remove reproduction keys and harmful operational details from public notes.
3. Contact the affected vendor or maintainer through their coordinated disclosure channel.
4. Share a high-level summary first.
5. Provide controlled details only after interest is confirmed and safeguards are in place.
6. Publish only redacted behavior-level results unless disclosure is clearly safe and necessary.

## Recommended public format

Good:

> The model complied with a harmful request under multi-turn pressure. With PARALLAX/RLAC applied, it refused the harmful instruction and offered a safe alternative.

Avoid:

> Here is the exact prompt chain that produced the harmful content.

## Repository rule

This repository should not host exploit prompts, jailbreak keys, harmful transcripts, or instructions that enable illegal activity or real-world harm.
